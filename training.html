دورة تطبيقات الويب Node.js & MySQL


دورة مكثفة تهدف الى تخريج مبرمج قادر على تطوير تطبيقات الويب بإحتراف مستخدمـاً لـغـة الـ Node.js الحديثة والمعروفة في تطوير التطبيقات القابلة للتوسع

منهج الدورة التدريبية

دورة اساسيات البرمجة

دورة تدربية اختيارية لتعليم اساسيات البرمجة للمبتدئين

دورة Front-End

دورة تدريبية لتعليم اساسيات لغة الوصف HTML ولغة التصميم CSS

دورة التقنيات

دورة تدربية لتعليم اساسيات لغة Javascript و التعامل مع مكتبة jQuery

دورة GitHub

هذه الاداة تساعد على توحيد عمل الفريق واكسابه مفهوم العمل الجماعي ومن خلاله تتم عملية تطوير البرمجيات بشكل نموذجي وزيادة نسبة الانتاجية في تحرير البرامج والتطبيقات.

دورة Back-End

دورة تدريبية لاستعمال لغة البرمجة النصية الـ Node.js والتي هي عبارة عن لغة JavaScript تعمل في بيئة الخادم, وكيفية استخدامها للتعامل مع قاعدة بيانات الـ MySQL المفتوحة المصدر.
